Text Styling in CSS

This is an example paragraph demonstrating text styling in CSS. You can modify various text properties such as font size, color, alignment, and more.

Here is a list of commonly used text properties in CSS:

  • font-size: Adjusts the size of the text.
  • color: Sets the text color.
  • text-align: Aligns text (left, right, center, or justify).
  • text-transform: Controls capitalization (uppercase, lowercase, capitalize).
  • letter-spacing: Adjusts spacing between letters.
  • line-height: Sets the height of a line of text.
  • text-decoration: Adds effects like underline, overline, or line-through.
